TP4 - Grupo Cantina Chichilo - 2do cuatrimestre 2025

Equipo con temática "gastronomía".

Este proyecto tiene el objetivo de rediseñar la página web "Cantina Chichilo", una cantina porteña de comida mayormente italiana. Se rediseñó toda la página web desde cero, dándole una diseño más atractivo, moderno e intuitivo para que los nuevos clientes puedan conocer todos los platos del menú, poder reservar con más facilidad.

Quisimos proponer que se combinara el ambiente italiano, que es cálido y familiero, con lo moderno y actual.
